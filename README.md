## 🎯 Обзор проекта

**Password Generator** — это быстрый клиентский инструмент для разработчиков, предназначенный для пакетной генерации устойчивых к взлому паролей. Приложение полностью работает без использования серверных мощностей (Zero-Backend Architecture), гарантируя абсолютную конфиденциальность обрабатываемых данных.

### Ключевые возможности:

* **Гибкая настройка символов:** Выборочное включение прописных (A-Z) и строчных (a-z) букв, цифровых рядов (0-9) и специальных символов (% @ # $ ! *), адаптирующееся под любые требования безопасности.
* **Пакетная генерация:** Мгновенное создание массивов данных объемом до 50 уникальных ключей за один клик.
* **Модификация «на лету»:** Автоматическое обновление результатов в режиме реального времени при любом изменении длины (от 4 до 64 символов) или пула используемых алфавитов.
* **Продвинутый UI/UX:** Анимированные элементы управления, микро-взаимодействия (копирование отдельных строк или всего массива в буфер) и адаптивная темная тема, снижающая нагрузку на зрение при ночной разработке.

## 🔒 Безопасность и Конфиденциальность

Проект использует нативное браузерное **Web Crypto API** (`crypto.getRandomValues()`) вместо стандартного метода `Math.random()`. Это гарантирует:

1. **Криптографическую стойкость** и защиту от предсказуемости генерируемых комбинаций.
2. **Локальное выполнение:** Логика работает строго в песочнице вашего браузера. Ни один сгенерированный байт не передается по сети, что исключает перехват или утечку паролей.

## 🚀 Быстрый запуск (Локальное окружение)

Так как проект не содержит тяжелых npm-зависимостей и этапа компиляции, развернуть его локально можно за пару шагов:

1. Склонируйте репозиторий:

```bash
git clone https://github.com/sarnerio/password-generator.git
cd password-generator

```

2. Запустите любой локальный статический сервер. Например, с помощью Python:

```bash
python3 -m http.server 8080

```

3. Откройте в браузере адрес: `http://localhost:8080`
